## 10. CMS 및 기술 운영 전략 (Tech Strategy)
> **"유연하고 강력한 미디어 엔진(Media Engine) 구축"**
> - *Opus 4.5가 설계하고 CSTO가 감수함.*

### 10.1 CMS 아키텍처 (V2.0)
- **Headless CMS 구조:** 프론트엔드(Next.js)와 데이터(Supabase)의 완벽한 분리.
- **데이터 구조화 (Schema-First):** `Categories`, `Menus`, `Layouts` 테이블을 통해 사이트 구조를 코딩 없이 DB에서 제어 가능.

### 10.2 자동화 센터 (Automation Center)
- **봇 스케줄러 (Scheduler):** 
    - 하루 3회 (09:00, 13:00, 17:00) 정기 실행 (Vercel Cron 연동).
    - 관리자 페이지(`/admin/bot/run`)에서 수동 실행 및 ON/OFF 제어 가능.
- **수집 범위 (Coverage):**
    - **광역:** 광주광역시, 전라남도청.
    - **기초:** 전남 22개 시/군 전체 (나주, 목포, 여수, 순천 등).
    - **교육:** 광주광역시 교육청, 전라남도 교육청 (별도 '교육' 카테고리 매핑).
- **기능(Features):**
    - **커스텀 링크(Custom Link):** 카테고리 메뉴 클릭 시 내부 목록이 아닌 외부 URL이나 특정 페이지로 이동 기능 지원.
    - **AI 재작성:** 수집된 보도자료를 AI가 '뉴스 스타일'로 자동 윤문.

### 10.3 지도 서비스 기술 (Map Tech)
- **엔진:** Naver Maps API (국내 최적화 지도 데이터).
- **데이터:** PostGIS를 활용한 위치 기반 쿼리 (내 주변 뉴스, 내 주변 맛집).
- **확장성:** 단순 핀(Marker) 표시를 넘어, 행정구역(Polygon) 오버레이 및 클러스터링 지원 예정.

### 10.4 관리자 경험 (Admin UX)
- **대시보드:** 봇 상태, 오늘의 수집 현황, 인기 기사(조회수)를 한눈에 파악.
- **콘텐츠 관리:** 기사 작성 시 AI 도구(요약, 맞춤법) 통합 제공.

- **관리자 기능:**
    - 이미지 업로드, 이동 링크(URL) 설정, 노출 위치(Left/Right), 노출 여부(ON/OFF) 제어.

### 11.2 데이터베이스 스키마 설계 (Draft)
```sql
CREATE TABLE ads (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  title TEXT NOT NULL,          -- 광고명 (관리용)
  position TEXT NOT NULL,       -- 위치: 'left_floating', 'right_floating', 'top_banner', 'article_bottom'
  image_url TEXT NOT NULL,      -- 배너 이미지 경로
  link_url TEXT,                -- 클릭 시 이동할 URL
  is_active BOOLEAN DEFAULT true,
  start_date TIMESTAMP WITH TIME ZONE, -- 게시 시작일
  end_date TIMESTAMP WITH TIME ZONE,   -- 게시 종료일
  sort_order INTEGER DEFAULT 0, -- 같은 위치 내 노출 순서
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

### 11.3 관리자 메뉴 구조 (Admin Menu)
- **위치:** `콘텐츠 관리` -> `광고/배너 관리` (신규 메뉴 추가)
- **기능:**
    - **광고 목록:** 현재 등록된 배너 리스트 확인 및 상태 변경.
    - **광고 등록:** 이미지 업로드 및 타겟 URL 설정.
    - **미리보기:** 실제 배너가 웹사이트에 어떻게 뜰지 미리 확인.
