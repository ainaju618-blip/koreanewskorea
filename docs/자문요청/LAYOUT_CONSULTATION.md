## 3. 자문 요청 사항

### 3.1 레이아웃 구조에 대한 질문

**Q1. 2컬럼 vs 3컬럼 레이아웃**
```
현재: 72% + 28% (2컬럼)

대안 A: 25% + 50% + 25% (3컬럼, 좌우 사이드바)
대안 B: 100% 풀와이드 (모바일 퍼스트)
대안 C: 현재 유지 (뉴스 사이트 표준)
```
- 뉴스 포털로서 어떤 레이아웃이 사용자 경험에 최적인가요?
- 광고 수익과 콘텐츠 집중도의 균형점은?

**Q2. 히어로 섹션 비율**
```
현재: 메인 이미지 7 : 사이드 리스트 5 (lg:grid-cols-12)

대안 A: 8 : 4 (메인 강조)
대안 B: 6 : 6 (균형)
대안 C: 풀와이드 슬라이더
```
- 첫 화면에서 뉴스 노출 개수 vs 단일 뉴스 집중도 중 무엇이 중요한가요?

**Q3. 지도 섹션 배치**
```
현재: 메인 콘텐츠 영역 내 (사이드바 옆)

대안 A: 히어로 바로 아래, 풀와이드
대안 B: 사이드바 내 작은 위젯
대안 C: 별도 /map 페이지로 분리
```
- 전국 지도가 메인 페이지에 필수인가요?
- 지역 선택 기능의 중요도는?
