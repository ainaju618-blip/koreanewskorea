## 📌 Executive Summary

코리아뉴스는 단순한 뉴스 포털이 아닌 **"공공 데이터 기반 지역 행정/생활 정보 허브"**로 포지셔닝한다. 네이버·다음과의 직접 경쟁을 피하고, **하이퍼로컬 거버넌스 플랫폼**이라는 블루오션을 개척한다.

### 핵심 전략 3축

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│   1️⃣ 공공 데이터 + 행정 서비스 통합                                          │
│      → 지자체 보도자료, 지원사업, 복지혜택 통합 제공                         │
│                                                                              │
│   2️⃣ 라이프 서비스 분리 운영                                                │
│      → 점술/운세는 서브도메인(fortune.koreanews...)으로 브랜드 분리          │
│                                                                              │
│   3️⃣ 신뢰 기반 하이퍼로컬 커뮤니티                                          │
│      → 지역 인증 + 클린 모더레이션으로 차별화                                │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 7.2 캐싱 전략

```typescript
// Next.js 15 캐싱 설정
export const revalidate = 600; // 10분

// 지역별 캐시 태그
export async function generateStaticParams() {
  const regions = await getAllRegions();
  return regions.map((region) => ({
    sido: region.code,
  }));
}

// On-demand Revalidation (기사 발행 시)
await revalidatePath(`/region/${sido}/${sigungu}`);
await revalidateTag(`news-${sido}`);
```
