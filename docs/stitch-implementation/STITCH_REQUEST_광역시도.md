## 2. 디자인 요구사항

### 2.1 페이지 성격
```
전국판 ←――――― 광역시도판 ―――――→ 시군판(나주)
(뉴스 포털)     (지역 뉴스 허브)     (행정/생활 밀착)
```

- **전국판보다**: 해당 시·도에 집중, 지역색 강조
- **시군판보다**: 뉴스 중심, 산하 시·군 연결 기능

### 2.2 필수 섹션

#### A. 헤더 영역
```
┌─────────────────────────────────────┐
│ ☰  KOREA NEWS 광주판    🔍 🔔 👤   │
│     Gwangju Edition                 │
├─────────────────────────────────────┤
│ 종합 | 정치 | 경제 | 사회 | 문화    │
└─────────────────────────────────────┘
```
- 지역명 강조 (예: "광주판", "경기판")
- 영문 서브타이틀 (예: "Gwangju Edition")
- 지역 대표 컬러 적용 가능

#### B. 히어로 섹션
```
┌─────────────────────────────────────┐
│  ┌─────────────────────────────┐   │
│  │      [히어로 뉴스 이미지]    │   │
│  │                             │   │
│  └─────────────────────────────┘   │
│  광주광역시, 2026년 예산안 확정    │
│  AI 산업 육성에 3,000억 투자       │
│  광주시청 · 정치 · 2시간 전        │
└─────────────────────────────────────┘
```
- 해당 시·도의 TOP 뉴스
- 출처에 시·도청/시·도의회 표시

#### C. 산하 시·군·구 바로가기 (핵심 차별점)
```
┌─────────────────────────────────────┐
│ 📍 광주광역시 5개 구                │
├─────────────────────────────────────┤
│ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐   │
│ │동구 │ │서구 │ │남구 │ │북구 │   │
│ └─────┘ └─────┘ └─────┘ └─────┘   │
│         ┌─────┐                    │
│         │광산구│                    │
│         └─────┘                    │
└─────────────────────────────────────┘
```
- 해당 광역시도의 시·군·구 목록
- 클릭 시 해당 지역 페이지로 이동
- 각 지역별 최신 뉴스 개수 뱃지 (선택)

#### D. 뉴스 리스트
```
┌─────────────────────────────────────┐
│ 🔥 실시간 주요 뉴스                 │
├─────────────────────────────────────┤
│ [시청] 광주시, 무료 와이파이 확대   │
│ [의회] 시의회 정례회 주요 안건      │
│ [경제] AI 기업 10개사 광주 이전     │
│ [문화] 비엔날레 사전 예약 시작      │
└─────────────────────────────────────┘
```
- 카테고리 뱃지 (시청/의회/경제/문화/사회)
- 썸네일 + 제목 + 요약 형식

#### E. 지역 통계/현황 위젯
```
┌─────────────────────────────────────┐
│ 📊 광주광역시 현황                  │
├───────────┬───────────┬─────────────┤
│ 인구      │ 날씨      │ 미세먼지    │
│ 145만명   │ 🌤 12°C  │ 좋음 😊     │
└───────────┴───────────┴─────────────┘
```
- 인구, 날씨, 대기질 등 기본 정보
- 공공 API 연동

#### F. 하단 네비게이션
```
┌─────────────────────────────────────┐
│  🏠    📍    ➕    📑    ☰        │
│  홈   시군구  제보  스크랩  전체    │
└─────────────────────────────────────┘
```
- "시군구" 버튼: 산하 지역 목록 모달

## 7. Stitch 프롬프트 예시

```
코리아뉴스 광역시도판 홈페이지를 디자인해주세요.

요구사항:
- 모바일 퍼스트 (max-w-md, 448px)
- 한국어 뉴스 포털
- 광주광역시를 예시로 작성

필수 섹션:
1. 헤더: 로고 "KOREA NEWS 광주판", 검색, 알림, 프로필
2. 카테고리 탭: 종합, 정치, 경제, 사회, 문화
3. 히어로 뉴스: 대표 이미지 + 제목 + 요약
4. 시·군·구 바로가기: 동구, 서구, 남구, 북구, 광산구 (5개 구)
5. 뉴스 리스트: 카테고리 뱃지 + 썸네일 + 제목
6. 지역 현황 위젯: 인구, 날씨, 미세먼지
7. 하단 네비게이션: 홈, 시군구, 제보, 스크랩, 전체

디자인 시스템:
- 폰트: Public Sans, Noto Sans KR
- 아이콘: Material Symbols Outlined
- Primary: #3c83f6
- 광주 액센트: #00A651 (녹색)

기존 디자인 참고:
- 전국판 메인 스타일 유지
- 나주판의 지역 밀착 요소 참고
```
