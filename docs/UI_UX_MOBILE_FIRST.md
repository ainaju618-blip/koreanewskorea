# 코리아뉴스 모바일 퍼스트 UI/UX 설계서

## 핵심 원칙

```
1. 신문사다움 - 관공서/일반인이 "신문사"로 인식
2. 모바일 퍼스트 - 스마트폰이 주 사용 환경
3. 뉴스 중심 - 맛집/여행은 "생활정보" 섹션
4. 지도 없음 - 버튼과 딥링크로 대체
5. 기존 데이터 활용 - 운영서버 자산 활용
```

---

## 계층 구조

| 단계 | 경로 | 성격 |
|------|------|------|
| 1단계 본사 | `/` | 전국 뉴스 허브, 지역 선택 |
| 2단계 광역시도 | `/region/gwangju` | 시도 뉴스, 시군 이동 |
| 3단계 시군 | `/region/naju` | 지역 뉴스 + 생활정보 |
