# 코리아NEWS 전국 계층 구조 (3단계 시스템)

> **Version:** 1.0
> **Created:** 2026-01-06
> **Status:** 시범 운영 중

## 1. 개요

코리아NEWS는 전국 17개 광역시도를 포괄하는 3단계 계층 구조로 설계되었습니다.

```
┌─────────────────────────────────────────────────────────────┐
│  1단계: 코리아NEWS 본사 (전국판)                             │
│         koreanewskorea.com                                  │
│         → 전국 17개 시도 뉴스 통합                           │
└─────────────────────────────────────────────────────────────┘
                              │
         ┌────────────────────┼────────────────────┐
         ▼                    ▼                    ▼
┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐
│ 2단계: 시도     │  │ 2단계: 시도     │  │ 2단계: 시도     │
│ (17개 광역시도) │  │ 예: 광주광역시  │  │ 예: 전라남도    │
│                 │  │                 │  │                 │
│ • 시도청 보도자료│  │ • 광주시청      │  │ • 전라남도청    │
│ • 시도의회      │  │ • 광주시의회    │  │ • 전라남도의회  │
│ • 시도교육청    │  │ • 광주교육청    │  │ • 전남교육청    │
└────────┬────────┘  └────────┬────────┘  └────────┬────────┘
         │                    │                    │
         ▼                    ▼                    ▼
┌─────────────────────────────────────────────────────────────┐
│ 3단계: 시군구 자치단체 (약 226개)                            │
│                                                             │
│ 예: 나주시, 진도군, 목포시, 순천시, 여수시 ...              │
│                                                             │
│ • 시군 자치단체 보도자료                                    │
│ • 시군 의회                                                 │
│ • 지역교육지원청                                            │
│ • 지역 생활뉴스                                             │
│ • 맛집/여행 정보 (카카오맵 연동)                            │
└─────────────────────────────────────────────────────────────┘
```

## 2. 현재 시범 운영 현황

### 2.1 완성/운영 중인 페이지

| 단계 | 지역 | URL 경로 | 상태 | 비고 |
|------|------|----------|------|------|
| **1단계** | 본사 (전국) | `/` | ✅ 운영 중 | Stitch 디자인 적용 |
| **2단계** | 광주광역시 | `/region/gwangju` | 🔨 개발 중 | 시범 2단계 |
| **3단계** | 나주시 | `/region/naju` | ✅ 운영 중 | 시범 3단계 |
| **3단계** | 진도군 | `/region/jindo` | ✅ 운영 중 | 시범 3단계 |

### 2.2 시범 구조도

```
koreanewskorea.com/                    ← 1단계: 본사 (전국판)
│
├── /region/gwangju/                   ← 2단계: 광주광역시 (시범)
│   ├── 광주시청 보도자료
│   ├── 광주시의회
│   ├── 광주교육청 보도자료
│   └── (하위 5개 구 - 미구현)
│
└── /region/jeonnam/                   ← 2단계: 전라남도 (예정)
    │
    ├── /region/naju/                  ← 3단계: 나주시 (시범)
    │   ├── 나주시청 보도자료
    │   ├── 나주시의회
    │   ├── 나주교육지원청
    │   ├── /food/                     맛집 목록
    │   ├── /food/[id]/                맛집 상세 (카카오맵)
    │   ├── /travel/                   여행 목록
    │   └── /travel/[id]/              여행 상세 (카카오맵)
    │
    └── /region/jindo/                 ← 3단계: 진도군 (시범)
        ├── 진도군청 보도자료
        ├── 진도군의회
        ├── 진도교육지원청
        ├── /food/                     맛집 목록 (예정)
        └── /travel/                   여행 목록 (예정)
```

## 3. 각 단계별 상세 구성

### 3.1 1단계: 본사 홈페이지 (전국판)

**URL:** `koreanewskorea.com/`

**구성 요소:**
```
┌─────────────────────────────────────────────────────────────┐
│ ZONE 0: MainHeroBanner (풀와이드 히어로 배너 + 검색)         │
├─────────────────────────────────────────────────────────────┤
│ ZONE 1: HeroSection                                         │
│ ┌───────────────────────────┬─────────────────────────┐     │
│ │ 메인뉴스 카드 (7컬럼)      │ 정책 브리핑 (5컬럼)     │     │
│ └───────────────────────────┴─────────────────────────┘     │
├─────────────────────────────────────────────────────────────┤
│ ZONE 2: MapSection                                          │
│ ┌───────────────────────────┬─────────────────────────┐     │
│ │ 전국 뉴스 지도             │ 실시간 뉴스 피드        │     │
│ └───────────────────────────┴─────────────────────────┘     │
├─────────────────────────────────────────────────────────────┤
│ ZONE 3: TravelSection (추천 여행지 4열 그리드)               │
├─────────────────────────────────────────────────────────────┤
│ ZONE 4: BranchPromoBanner (지사 안내 배너)                   │
└─────────────────────────────────────────────────────────────┘
```

**컴포넌트:**
- `MainHeroBanner` - 히어로 배너
- `HeroSection` - 메인 뉴스 + 정책 브리핑
- `MapSection` - 전국 지도 + 실시간 피드
- `TravelSection` - 추천 여행지

**데이터 소스:**
- 17개 시도 통합 뉴스 피드
- 전국 주요 정책 브리핑

---

### 3.2 2단계: 광역시도 홈페이지

**URL 패턴:** `/region/{sido}`
**시범:** `/region/gwangju` (광주광역시)

**보도자료 소스 (3종):**

| 구분 | 소스 | 예시 (광주) |
|------|------|------------|
| 시도 지자체 | 시도청 보도자료 | 광주광역시청 |
| 시도 의회 | 시도의회 보도자료 | 광주광역시의회 |
| 시도 교육청 | 시도교육청 보도자료 | 광주광역시교육청 |

**구성 요소:**
```
┌─────────────────────────────────────────────────────────────┐
│ 지역 헤더 (지역명, 슬로건, 날씨)                             │
├─────────────────────────────────────────────────────────────┤
│ 1. 시도 지자체 보도자료 섹션                                 │
│    └── 광주시청 보도자료                                    │
├─────────────────────────────────────────────────────────────┤
│ 2. 시도 의회 보도자료 섹션                                   │
│    └── 광주시의회 보도자료                                  │
├─────────────────────────────────────────────────────────────┤
│ 3. 시도 교육청 보도자료 섹션                                 │
│    └── 광주교육청 보도자료                                  │
├─────────────────────────────────────────────────────────────┤
│ 4. 하위 시군구 뉴스 통합 피드 (해당 시 5개 구)               │
├─────────────────────────────────────────────────────────────┤
│ 5. 지역 특화 콘텐츠 (맛집, 여행, 행사)                       │
└─────────────────────────────────────────────────────────────┘
```

**스크래퍼 소스:**
- `scrapers/gwangju/` - 광주시청 보도자료
- `scrapers/gwangju_council/` - 광주시의회 (예정)
- `scrapers/gwangju_edu/` - 광주교육청 보도자료

### 3.3 3단계: 시군구 자치단체 홈페이지

**URL 패턴:** `/region/{sigungu}` 또는 `/region/{sido}/{sigungu}`
**시범:** `/region/naju` (나주시), `/region/jindo` (진도군)

**보도자료 소스 (3종):**

| 구분 | 소스 | 예시 (나주) |
|------|------|------------|
| 시군 지자체 | 시군청 보도자료 | 나주시청 |
| 시군 의회 | 시군의회 보도자료 | 나주시의회 |
| 지역 교육지원청 | 교육지원청 보도자료 | 나주교육지원청 |

**구성 요소:**
```
┌─────────────────────────────────────────────────────────────┐
│ 지역 헤더 (지역명, 슬로건 배지, 날씨)                        │
├─────────────────────────────────────────────────────────────┤
│ 1. 시군 자치단체 보도자료                                    │
│    └── 나주시청/진도군청 보도자료                           │
├─────────────────────────────────────────────────────────────┤
│ 2. 시군 의회 보도자료                                        │
│    └── 나주시의회/진도군의회 보도자료                       │
├─────────────────────────────────────────────────────────────┤
│ 3. 지역 교육지원청 보도자료                                  │
│    └── 나주교육지원청/진도교육지원청 보도자료               │
├─────────────────────────────────────────────────────────────┤
│ 4. 지역 생활뉴스                                            │
├─────────────────────────────────────────────────────────────┤
│ 5. 맛집 정보 (/food)                                        │
│    └── 상세 페이지에 카카오맵 연동                          │
├─────────────────────────────────────────────────────────────┤
│ 6. 여행 정보 (/travel)                                      │
│    └── 상세 페이지에 카카오맵 연동                          │
├─────────────────────────────────────────────────────────────┤
│ 7. 지역 행사/이벤트                                         │
└─────────────────────────────────────────────────────────────┘
```

**스크래퍼 소스 (나주 예시):**
- `scrapers/naju/` - 나주시청 보도자료
- `scrapers/naju_council/` - 나주시의회 보도자료
- `scrapers/jeonnam_edu/` - 전남교육청 (나주교육지원청 포함)

**스크래퍼 소스 (진도 예시):**
- `scrapers/jindo/` - 진도군청 보도자료
- `scrapers/jindo_council/` - 진도군의회 보도자료
- `scrapers/jeonnam_edu/` - 전남교육청 (진도교육지원청 포함)

## 4. 보도자료 소스 체계 정리

### 4.1 단계별 소스 구성

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 1단계: 본사 (전국판)                                                    │
│ └── 17개 시도 통합 뉴스 피드                                            │
├─────────────────────────────────────────────────────────────────────────┤
│ 2단계: 광역시도 (17개)                                                  │
│ ├── ① 시도청 보도자료 (예: 광주광역시청)                               │
│ ├── ② 시도의회 보도자료 (예: 광주광역시의회)                           │
│ └── ③ 시도교육청 보도자료 (예: 광주광역시교육청)                       │
├─────────────────────────────────────────────────────────────────────────┤
│ 3단계: 시군구 (약 226개)                                                │
│ ├── ① 시군청 보도자료 (예: 나주시청)                                   │
│ ├── ② 시군의회 보도자료 (예: 나주시의회)                               │
│ └── ③ 지역교육지원청 보도자료 (예: 나주교육지원청)                     │
└─────────────────────────────────────────────────────────────────────────┘
```

### 4.2 교육청 계층 구조

```
시도교육청 (17개)
└── 지역교육지원청 (시군 단위)
    ├── 예: 전라남도교육청
    │   ├── 나주교육지원청
    │   ├── 진도교육지원청
    │   ├── 목포교육지원청
    │   └── ... (22개 시군)
    │
    └── 예: 광주광역시교육청
        ├── 동부교육지원청
        ├── 서부교육지원청
        └── ... (5개 구)
```
