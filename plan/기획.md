### 📋 콘텐츠 생산 구조 (확정)

#### 전체 구조도
```
┌─────────────────────────────────────────────────────────────┐
│                    콘텐츠 생산 파이프라인                      │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  [자동수집] ──→ [AI분류] ──→ [기자배정] ──→ [발행]           │
│      │              │            │                          │
│      ↓              ↓            ↓                          │
│   스크래퍼      카테고리      바이라인                       │
│   15-20개/일   시정/의회/    (지역=코리아뉴스)               │
│               교육/생활      홍길동 기자                     │
│                                                              │
├─────────────────────────────────────────────────────────────┤
│  [오피니언] ──→ 유일한 자체생산 (칼럼, 사설)                  │
├─────────────────────────────────────────────────────────────┤
│  [문화섹션] ──→ 공공데이터 API (한국관광공사, 문화재청)        │
│               여행/맛집/문화유적                             │
└─────────────────────────────────────────────────────────────┘
```

#### 카테고리별 상세

| 카테고리 | 콘텐츠 소스 | 외부 인식 | 상세 설명 |
|----------|------------|----------|----------|
| 시정 | 자동수집 + AI분류 | 자체 취재 | 시청 보도자료, 정책 뉴스 |
| 의회 | 자동수집 + AI분류 | 자체 취재 | 시의회, 도의회 소식 |
| 교육 | 자동수집 + AI분류 | 자체 취재 | 교육청, 학교 소식 |
| 생활 | 자동수집 + AI분류 | 자체 취재 | 일반 생활/사회 뉴스 |
| **오피니언** | **유일한 자체생산** | 자체 취재 | 칼럼, 사설, 기고 |
| 문화 | 공공데이터 API | 자체 취재 | 여행/맛집/문화유적 |

#### 기자 바이라인 자동 배정 시스템

주인님 발언:
> "하루 15-20개의 기사가 입력되면 자동으로 카테고리로 분류되는 분류형 메뉴는 어때?"

**동작 방식**:
1. 스크래퍼가 기사 수집 (15-20개/일)
2. AI가 카테고리 자동 분류 (시정/의회/교육/생활)
3. 발행 시 담당 기자 자동 배정
4. 바이라인 생성: `(나주=코리아뉴스) 홍길동 기자`

**외부 인식**: "전문 기자가 직접 취재한 기사"

### 🍔 메뉴 구조: B안 선택

#### A안 vs B안 비교

| 구분 | A안 | B안 (선택) |
|------|-----|-----------|
| 스타일 | 포털형 | 신문사형 |
| 메뉴 구조 | 동적 카테고리 | 고정 카테고리 |
| 문화 메뉴 | 별도 아이콘 메뉴 | 드롭다운 |
| 느낌 | 네이버 뉴스 | 조선일보/중앙일보 |

#### B안 메뉴 구조 (확정)

```
┌─────────────────────────────────────────────────────────────┐
│  시정 │ 의회 │ 교육 │ 생활 │ 오피니언 │ 문화▼              │
│                                         ├─ 여행             │
│                                         ├─ 맛집             │
│                                         └─ 문화유적         │
└─────────────────────────────────────────────────────────────┘
```

**선택 이유**:
- 신문사 표준 카테고리 메뉴
- 문화는 드롭다운 (공공데이터 기반 콘텐츠 분리)
- 오피니언 강조 (유일한 자체생산)
- 광고주에게 "종합일간지" 인식
