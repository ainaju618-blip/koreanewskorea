## 1. 프로젝트 개요

### 1.1 목적
세계 주요 AI 뉴스 매체의 기사를 자동 수집하여 한국어로 재구성, 독자적인 AI 뉴스 콘텐츠 생산

### 1.2 핵심 전략

| 전략 | 용도 | 저작권 |
|------|------|--------|
| **전략 A: 참조용** | 원문 소개 + 링크 제공 | Fair Use (안전) |
| **전략 B: 재구성** | 사실 추출 → AI 재작성 | Transformative (안전) |

### 1.3 기술 스택

| 구분 | 기술 |
|------|------|
| 스크래핑 | Python (Playwright/BeautifulSoup) |
| RSS 파싱 | feedparser |
| 번역 | Google Translate (무료) |
| AI 재작성 | OpenAI GPT-4o |
| 저장소 | Supabase (PostgreSQL) |
| 프론트엔드 | Next.js 15 |

## 3. 수집 전략

### 3.1 전략 A: 참조용 (Reference Mode)

**목적**: 원문 소개, 트래픽 유도, 큐레이션

**출력 형식**:
```
[제목] OpenAI, GPT-5 발표
[요약] 1-2문장 요약
[원문] https://techcrunch.com/...
[더 읽기] 버튼
```

**저작권**: Fair Use - 안전
- 제목 + 짧은 요약 + 링크만 제공
- 썸네일 사용 안함 (또는 자체 생성)

### 3.2 전략 B: 재구성 (Transformative Mode)

**목적**: 독자적 콘텐츠 생산, 한국 독자 맞춤

**핵심 원칙**: "Facts are Free"
- 사실(Fact)은 저작권 보호 대상 아님
- 표현(Expression)만 저작권 적용
- 사실만 추출 → 새로운 표현으로 재작성

**변환 파이프라인**:
```
[원문 수집] → [사실 추출] → [번역] → [AI 재작성] → [검수] → [발행]
```

**사실 추출 구조**:
```json
{
  "who": "OpenAI",
  "what": "GPT-5 발표",
  "when": "2025-01-15",
  "where": "샌프란시스코",
  "why": "AI 성능 향상",
  "how": "추론 능력 강화",
  "numbers": ["100배 빠름", "1조 파라미터"],
  "quotes": []
}
```

## 4. 데이터 플로우

### 4.1 수집 단계

```
[외부 소스]
     │
     ▼
┌─────────────────────────────────────┐
│ 1. 수집 (Collector)                  │
│    - RSS: feedparser                │
│    - Web: Playwright                │
│    - 주기: 1시간마다                  │
└─────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────┐
│ 2. 중복 체크 (Deduplicator)          │
│    - URL 해시 비교                   │
│    - 제목 유사도 검사                 │
└─────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────┐
│ 3. 원문 저장 (Raw Storage)           │
│    - 테이블: ai_news_raw             │
│    - 원문 전체 보관 (법적 증빙)        │
└─────────────────────────────────────┘
```

### 4.2 가공 단계

```
┌─────────────────────────────────────┐
│ 4. 번역 (Translator)                 │
│    - Google Translate 무료 API       │
│    - 영문 → 한글                     │
└─────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────┐
│ 5. 사실 추출 (Fact Extractor)        │
│    - GPT-4o 사용                     │
│    - 5W1H 구조화                     │
└─────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────┐
│ 6. AI 재작성 (Rewriter)              │
│    - GPT-4o 사용                     │
│    - 새로운 문장으로 작성             │
│    - 한국 독자 관점 추가              │
└─────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────┐
│ 7. 품질 검사 (Quality Check)         │
│    - 원문 유사도 30% 이하 확인        │
│    - 팩트 정확성 검증                 │
└─────────────────────────────────────┘
     │
     ▼
┌─────────────────────────────────────┐
│ 8. 발행 (Publisher)                  │
│    - posts 테이블 저장               │
│    - 카테고리: AI                    │
│    - 상태: draft → 검수 → published  │
└─────────────────────────────────────┘
```
