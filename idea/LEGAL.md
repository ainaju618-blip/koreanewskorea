# 법적 고려사항

## 1. 핵심 법적 원칙

### 1.1 "Facts Are Free" 원칙

```
┌─────────────────────────────────────────────────────────────┐
│  저작권법의 핵심 구분                                        │
│                                                              │
│  ✅ 저작권 보호 대상 아님:                                   │
│     - 사실(Fact) 자체                                        │
│     - 아이디어(Idea)                                         │
│     - 데이터/정보                                            │
│                                                              │
│  ❌ 저작권 보호 대상:                                        │
│     - 사실의 표현 방식 (Expression)                          │
│     - 문장 구조와 서술                                       │
│     - 창작적 편집/배열                                       │
│                                                              │
│  → 사실만 추출하고 새롭게 표현하면 저작권 문제 없음           │
└─────────────────────────────────────────────────────────────┘
```

**법적 근거:**
- 미국: Feist Publications v. Rural Telephone (1991)
- 한국: 저작권법 제7조 (사실의 전달에 불과한 시사보도는 저작물 아님)

## 2. 전략별 법적 분석

### 2.1 전략 A: 참조용 (Reference Mode)

**방식:**
- 제목 + 1-2문장 요약 + 원문 링크 제공
- 썸네일 사용 안 함 (또는 자체 생성)

**법적 평가:**
| 항목 | 평가 | 비고 |
|------|------|------|
| Fair Use | ✅ 안전 | 비상업적 정보 제공 목적 |
| 인용 | ✅ 적법 | 정당한 범위 내 인용 |
| 썸네일 | ⚠️ 주의 | 직접 링크 피할 것 |

**권장 형식:**
```
[제목] OpenAI, GPT-5 발표
[요약] OpenAI가 차세대 AI 모델을 공개했다.
[원문] TechCrunch - 기사 보기 →
```

**위험도:** ⭐ (매우 낮음)

---

### 2.2 전략 B: 재구성 (Transformative Mode)

**방식:**
- 원문에서 사실만 추출
- AI가 완전히 새로운 문장으로 재작성
- 원문 유사도 30% 이하 유지

**법적 평가:**
| 항목 | 평가 | 비고 |
|------|------|------|
| Transformative Use | ✅ 안전 | 새로운 창작물 생성 |
| 사실 추출 | ✅ 적법 | 사실은 저작권 보호 대상 아님 |
| 표현 방식 | ✅ 적법 | 완전히 새로운 표현 사용 |
| 출처 표기 | ✅ 권장 | 신뢰도 + 윤리적 준수 |

**필수 조건:**
1. 원문 문장 그대로 사용 금지
2. 문장 구조 복제 금지
3. 독창적 표현으로 재서술
4. 사실 정확성 유지
5. 출처 명시 (윤리적 의무)

**위험도:** ⭐⭐ (낮음, 조건 준수 시)

## 3. 위험 요소 및 대응

### 3.1 잠재적 위험

| 위험 | 발생 조건 | 대응 방안 |
|------|----------|----------|
| 표절 시비 | 유사도 30% 초과 | 품질 검사 강화, 재작성 |
| 이미지 저작권 | 원본 이미지 사용 | 자체 이미지 생성 또는 미사용 |
| 상표권 침해 | 로고 무단 사용 | 로고 사용 금지 |
| 명예훼손 | 사실 왜곡 | 팩트 검증 강화 |

### 3.2 안전 장치

```python
# 품질 검사 기준
SIMILARITY_THRESHOLD = 0.30  # 30% 이하 필수

def quality_check(original: str, rewritten: str) -> bool:
    similarity = calculate_similarity(original, rewritten)
    if similarity > SIMILARITY_THRESHOLD:
        return False  # 재작성 필요
    return True
```

## 6. 국가별 법률 차이

### 6.1 미국 (주요 수집처 소재지)

**Fair Use 4가지 요소:**
1. 사용 목적 (상업적 vs 비상업적)
2. 저작물의 성격 (사실적 vs 창작적)
3. 사용 분량
4. 시장에 미치는 영향

**판단:** 사실 기반 재작성은 Transformative Use로 인정 가능

### 6.2 한국 (서비스 제공지)

**저작권법 제28조 (공표된 저작물의 인용):**
- 보도, 비평, 교육, 연구 목적
- 정당한 범위 안에서 공정한 관행

**판단:** 출처 명시 + 정당한 범위 인용 시 적법
