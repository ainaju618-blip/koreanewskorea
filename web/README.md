# web/ 폴더

> **용도**: 공개 정적 리소스 저장소

## 구조

```
web/
└── public/
    └── images/
        └── news/           # 뉴스 썸네일 이미지 (지역별)
```

## 참고

- 실제 Next.js 애플리케이션 소스는 프로젝트 루트의 `src/` 폴더에 위치
- 이 폴더는 추가 정적 리소스 저장용으로만 사용
- 주요 public 리소스는 루트의 `public/` 폴더 사용
