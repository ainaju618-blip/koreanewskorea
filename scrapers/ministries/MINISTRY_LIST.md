## 5. Common Patterns

### URL Patterns
```
Most ministries use these patterns:
- /bbs/list.do
- /board/list.do
- /news/press.do
- /selectBoardList.do
```

### Pagination
```
- pageIndex=1, 2, 3...
- page=1, 2, 3...
- nPage=1, 2, 3...
```

### Common CMS Systems
- eGovFrame (Korean government standard)
- Custom CMS per ministry
- Some use WYSIWYG editors
