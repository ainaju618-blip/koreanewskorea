# 🔮 주역 + 심리코칭 융합 서비스 설계 v2

## 📌 핵심 컨셉

```
"전통 주역의 깊이" + "현대 심리학의 공감" = 마음이 치유되는 운세
```

---

## 🎲 점술 방식 선택 (4가지)

### 점술 알고리즘 비교

| 방식 | 설명 | 변효 | 확률 분포 | 소요시간 | 권장 대상 |
|------|------|------|----------|---------|----------|
| **정통 384효** | 조주역학회 전통 방식 | 없음 (1효만) | 균등 1/384 | ~3분 | 빠른 답변 원할 때 |
| **동전점** | 동전 3개×6회 | 있음 (0~6개) | 비균등 | ~5분 | 일반 사용자 |
| **시초점** | 서죽 50개 전통 방식 | 있음 (0~6개) | 비균등 | ~20분 | 깊은 해석 원할 때 |
| **운발수 적용** | 기본 균등 + 가중치 | 없음 | 가중 균등 | 즉시 | 기본값 |

### 🔮 정통 384효 점술법 (조주역학회 방식)

**특징**: 3번의 조작으로 64괘 × 6효 = 384효 중 1개 직접 결정

**절차**:
```
┌─────────────────────────────────────────────────────────────┐
│  1️⃣ 축원 (마음 집중)                                        │
│     "천하언재시리오 지하언재시리니                           │
│      고지즉응하시고 감위수통하소서"                          │
├─────────────────────────────────────────────────────────────┤
│  2️⃣ 하괘 결정                                               │
│     49개 서죽 → 둘로 나눔 → 왼손 취함                        │
│     (태극1 + 왼손) ÷ 8 = 나머지 → 선천팔괘 번호              │
├─────────────────────────────────────────────────────────────┤
│  3️⃣ 상괘 결정                                               │
│     같은 과정 반복 → 64괘 결정                               │
├─────────────────────────────────────────────────────────────┤
│  4️⃣ 효 결정                                                 │
│     (태극1 + 왼손) ÷ 6 = 나머지 → 효 위치 (1~6)              │
└─────────────────────────────────────────────────────────────┘
```

**선천팔괘 순서**:
```
1:건(乾)☰  2:태(兌)☱  3:리(離)☲  4:진(震)☳
5:손(巽)☴  6:감(坎)☵  7:간(艮)☶  8:곤(坤)☷
```

**장점**:
- ✅ 빠른 진행 (3번 조작)
- ✅ 균등 확률 (모든 효 1/384)
- ✅ 단일 효사 해석 (명확한 답변)

**단점**:
- ❌ 변효가 없어 지괘(변화 후 괘) 없음
- ❌ 복합적 해석 어려움

---

### 🪙 동전점 (육효점)

**특징**: 동전 3개를 6번 던져 6효 구성

**효 값 계산**:
```
앞(1) + 앞(1) + 앞(1) = 3 → 노양(9) 변효 ⚡
앞(1) + 앞(1) + 뒤(0) = 2 → 소음(8)
앞(1) + 뒤(0) + 뒤(0) = 1 → 소양(7)
뒤(0) + 뒤(0) + 뒤(0) = 0 → 노음(6) 변효 ⚡
```

**확률 분포**:
- 노양(9): 1/8 = 12.5%
- 소음(8): 3/8 = 37.5%
- 소양(7): 3/8 = 37.5%
- 노음(6): 1/8 = 12.5%

---

### 🌿 시초점 (서죽점)

**특징**: 50개 서죽으로 18번 조작 (가장 정통)

**확률 분포** (전통):
- 노음(6): 1/16 ≈ 6.25%
- 소양(7): 5/16 ≈ 31.25%
- 소음(8): 7/16 ≈ 43.75%
- 노양(9): 3/16 ≈ 18.75%

---

### 📊 변효 해석 규칙 (주희 본의)

| 변효 수 | 해석 방법 |
|--------|----------|
| 0개 | 본괘 괘사(卦辭) |
| 1개 | 해당 효사(爻辭) |
| 2개 | 두 변효 중 상위 효사 |
| 3개 | 본괘 + 지괘 괘사 |
| 4개 | 지괘의 불변효 중 하위 효사 |
| 5개 | 지괘의 유일 불변효 효사 |
| 6개 | 지괘 괘사 (건괘=용구, 곤괘=용육) |

## 📋 답변 구조 설계 (5단계)

### 전체 흐름

```
┌─────────────────────────────────────────────────────────────┐
│  1️⃣ [점괘 읽기] 그대의 마음을 살피노라                      │
│     → 질문에서 숨겨진 마음 파악, 점술적 공감                │
├─────────────────────────────────────────────────────────────┤
│  2️⃣ [괘상] 주역이 보여주는 상(象)                           │
│     → 본괘 + 지괘 시각화, 괘 이름/의미                      │
├─────────────────────────────────────────────────────────────┤
│  3️⃣ [점사] 괘사와 효사                                      │
│     → 원문 + 풀이, 권위감/신뢰감                            │
├─────────────────────────────────────────────────────────────┤
│  4️⃣ [운세 풀이] 길흉 판단                                   │
│     → 대길~대흉, 카테고리별 상세 해석                       │
├─────────────────────────────────────────────────────────────┤
│  5️⃣ [점괘의 가르침] 하늘이 이르는 바                        │
│     → 심리 유형 맞춤 해석, 실천 지침, 마무리                │
└─────────────────────────────────────────────────────────────┘
```

## 📝 단계별 상세 설계

### 1️⃣ 점괘 읽기 (Reading the Seeker)

**목적**: 점을 치러 온 사람의 마음을 읽어주는 느낌

**구성**:
```
🔮 점괘를 풀어보노라

"{질문}"

{심리분석_결과}
→ 그대는 지금 {숨겨진_욕구}을/를 구하고 있도다.
```

**예시**:
```
🔮 점괘를 풀어보노라

"비트코인 지금 사도 될까요?"

점괘를 보니, 마음의 방향이 이미 정해져 있구나.
→ 그대는 '확신'을 구하고 있도다.
   결심은 섰으나, 누군가 "좋다"고 말해주길 바라는 것이로다.
```

### 3️⃣ 괘사/효사 (Traditional Interpretation)

**목적**: 전통의 권위, 깊이감

**구성**:
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📖 주역의 말씀
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【괘사(卦辭)】
"乾 元亨利貞"
 건은 크게 형통하니, 바르게 함이 이롭다.

【효사(爻辭)】- 상효
"亢龍有悔"
 높이 오른 용은 후회가 있으리라.

【풀이】
용이 너무 높이 올라 더 이상 오를 곳이 없으니,
지나침은 모자람만 못하다는 가르침입니다.
지금은 전진보다 '지킴'이 필요한 때입니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**효사 표기 규칙**:
```
양효(陽爻): 초구, 구이, 구삼, 구사, 구오, 상구
음효(陰爻): 초육, 육이, 육삼, 육사, 육오, 상육

변효 해석 규칙 (주희 주역본의):
- 0변효 → 본괘 괘사
- 1변효 → 해당 효사
- 2변효 → 상위 효사
- 3변효 → 본괘+지괘 괘사
- 4변효 → 지괘의 불변효 중 하위
- 5변효 → 지괘의 유일 불변효
- 6변효 → 지괘 괘사 (건/곤 특수)
```

### 5️⃣ 점괘의 가르침 (Oracle's Teaching)

**목적**: 점술적 마무리, 실천 지침, 여운

**구성**:
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

{심리유형_맞춤_해석}

【확인을 구하는 그대에게】

그대의 마음은 이미 답을 품고 있도다.
그 직감이 곧 하늘의 뜻이니라.

허나, 점괘가 이르기를,
"한 번에 전부"보다 "나누어 천천히"가 이롭도다.

   • 잃어도 무방한 만큼만 움직여라

그대의 길을 따라가도 좋으리라.
하늘이 그대의 앞길을 열어주리라. 🌟

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
