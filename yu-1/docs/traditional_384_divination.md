# 정통 384효 점술법 (傳統 三百八十四爻 占術法)

> 조주역학회를 비롯한 전통 역학 단체에서 사용하는 정통 기법

---

## 개요

384효 점술법은 주역 64괘의 각 괘에 6개의 효가 있어 총 384개의 효사(爻辭) 중 하나를 선택하는 점술 방법입니다.

```
64괘 × 6효 = 384효
```

---

## 준비물

- **서죽(筮竹)** 또는 **산가지**: 대나무 50개
- 경건한 마음

---

## 점술 절차

### 1단계: 태극(太極) 설정 및 축원

#### 1-1. 태극 설정
50개의 서죽 중 **1개를 바닥에 내려놓습니다**.
이것은 **태극(太極)** 을 상징합니다.

#### 1-2. 축원문 낭송

```
천하언재시리오 지하언재시리니
고지즉응하시고 감위수통하소서.

건명(乾命) [이름], 하늘에 뜻을 구합니다.
[질문 내용]
```

**축원문 해석:**
- 천하언재시리오(天何言哉是理오): 하늘이 무슨 말씀을 하시겠습니까, 이치가 있으니
- 지하언재시리니(地何言哉是理니): 땅이 무슨 말씀을 하시겠습니까, 이치가 있으니
- 고지즉응하시고(告之則應하시고): 고하면 응하시고
- 감위수통하소서(感爲遂通하소서): 느끼시어 통하게 해주소서

---

### 2단계: 하괘(下卦) 결정

```
[나머지 49개 서죽]
       ↓
[랜덤하게 둘로 나눔]
       ↓
┌──────────┬──────────┐
│   왼손   │   오른손  │
│  (취함)  │  (버림)   │
└──────────┴──────────┘
       ↓
[태극(1개) + 왼손 서죽 = 합계]
       ↓
[합계 ÷ 8의 나머지 = 하괘]
```

**나머지 처리:**
- 나머지가 1~7이면 그대로 사용
- 나머지가 **0이면 8**로 처리 (곤괘)

---

### 3단계: 상괘(上卦) 결정

2단계와 동일한 과정을 반복합니다.

```
[49개를 다시 랜덤하게 둘로 나눔]
       ↓
[왼손 취함, 오른손 버림]
       ↓
[태극(1개) + 왼손 서죽 = 합계]
       ↓
[합계 ÷ 8의 나머지 = 상괘]
```

---

### 4단계: 효(爻) 결정

```
[49개를 다시 랜덤하게 둘로 나눔]
       ↓
[왼손 취함, 오른손 버림]
       ↓
[태극(1개) + 왼손 서죽 = 합계]
       ↓
[합계 ÷ 6의 나머지 = 효 위치]
```

**나머지 처리:**
- 나머지가 1~5이면 그대로 사용
- 나머지가 **0이면 6**으로 처리 (상효)

## 64괘 구성

상괘와 하괘의 조합으로 64괘가 만들어집니다.

```
┌─────────┐
│  상괘   │ ← 4~6효
├─────────┤
│  하괘   │ ← 1~3효
└─────────┘
```

### 예시: 상괘=6(감), 하괘=3(리)

```
상괘 감(坎) ☵ = 음양음
하괘 리(離) ☲ = 양음양

결과:
  6효: === === (음)  ┐
  5효: ======= (양)  ├ 상괘: 坎(물)
  4효: === === (음)  ┘
  3효: ======= (양)  ┐
  2효: === === (음)  ├ 하괘: 離(불)
  1효: ======= (양)  ┘

→ 제64괘 화수미제(火水未濟)
```

## 결과 해석

### 점괘 결과
```
상괘(1~8) + 하괘(1~8) → 64괘 중 1괘 결정
효 위치(1~6) → 해당 괘의 특정 효사(爻辭) 참조
```

### 해석 순서
1. **괘사(卦辭)**: 해당 괘 전체의 의미
2. **효사(爻辭)**: 선택된 효의 구체적 의미
3. **상전(象傳)**: 괘상의 상징적 해석

## 확률 분포

이 방법은 **균등 분포**에 가깝습니다.

### 이론적 확률
- 각 괘 출현 확률: 1/64 ≈ 1.56%
- 각 효 출현 확률: 1/6 ≈ 16.67%
- 각 효사 출현 확률: 1/384 ≈ 0.26%

### 확률 분석
```
49개를 반으로 나눌 때:
- 왼손: 1~48개 (균등 분포)
- 태극 합산: 2~49
- 8로 나눈 나머지: 0~7 (거의 균등)
- 6으로 나눈 나머지: 0~5 (거의 균등)
```
