### 1.1 서비스 소개

**핵심 컨셉**:
```
"전통 주역의 깊이" + "현대 심리학의 공감" = 마음이 치유되는 운세
```

**차별점**:
- 일반 운세앱: "대길입니다" (단순 결과)
- 우리 서비스: 괘상 + 괘사 + 효사 + 심리분석 + 맞춤조언 (깊이 있는 해석)

## 2. 제작 요청 영상 (3종)

### 2.1 🪙 동전점 (육효점) 애니메이션

**점술 방식 설명**:
- 동전 3개를 6번 던져 괘(卦)를 구성
- 각 던지기마다 앞/뒤 조합으로 음효/양효 결정
- 소요시간: 약 5분
- 대상: 일반 사용자

**효 값 계산 원리**:
```
앞(1) + 앞(1) + 앞(1) = 3 → 노양(老陽) 변효 ⚡
앞(1) + 앞(1) + 뒤(0) = 2 → 소음(少陰)
앞(1) + 뒤(0) + 뒤(0) = 1 → 소양(少陽)
뒤(0) + 뒤(0) + 뒤(0) = 0 → 노음(老陰) 변효 ⚡
```

**영상 컨셉**: "우주 속 운명의 동전"

**시각 요소**:
| 요소 | 설명 |
|------|------|
| 배경 | 심연의 우주 (보라/남색 네뷸라 + 별빛) |
| 주요 오브젝트 | 황금빛 고대 동전 3개 (음양 문양) |
| 동작 | 동전이 천천히 회전하며 떠다님 |
| 효과 | 동전 주위 태극 문양 빛 궤적 |
| 특수효과 | 동전 던질 때 별똥별처럼 빛 산란 |
| 색감 | 금색 동전 + 보라색 네뷸라 + 짙은 남색 우주 |

**영상 사양**:
- 길이: 15-30초 (심리스 루프)
- 해상도: 1920x1080 (16:9) 또는 1080x1920 (9:16 모바일)
- 형식: MP4/WebM (웹 최적화)

### 2.2 🌿 시초점 (서죽점) 애니메이션

**점술 방식 설명**:
- 50개의 서죽(대나무 막대)으로 18번 조작
- 가장 정통적인 주역 점술법
- 소요시간: 약 20분
- 대상: 깊은 해석을 원하는 사용자

**확률 분포 (전통)**:
```
노음(6): 1/16 ≈ 6.25%
소양(7): 5/16 ≈ 31.25%
소음(8): 7/16 ≈ 43.75%
노양(9): 3/16 ≈ 18.75%
```

**영상 컨셉**: "우주의 서죽 (시간의 강)"

**시각 요소**:
| 요소 | 설명 |
|------|------|
| 배경 | 은하수 + 별의 흐름 (시간의 강) |
| 주요 오브젝트 | 50개의 빛나는 대나무 막대 (청록빛) |
| 동작 | 서죽이 은하수 흐름을 따라 유영 |
| 효과 | 서죽 나눌 때 팔괘(八卦) 문양 출현 |
| 특수효과 | 수 카운팅 시 숫자가 별처럼 반짝 |
| 색감 | 청록색 서죽 + 은하수 보라/파랑 + 금색 문양 |

**영상 사양**:
- 길이: 30-60초 (심리스 루프)
- 해상도: 1920x1080 / 1080x1920
- 형식: MP4/WebM

### 2.3 🔮 정통 384효 애니메이션

**점술 방식 설명**:
- 조주역학회(調周易學會) 전통 방식
- 3번의 조작으로 64괘 × 6효 = 384효 중 1개 직접 결정
- 소요시간: 약 3분
- 특징: 균등 확률 (1/384), 빠른 답변

**절차**:
```
1️⃣ 축원 (마음 집중)
   "천하언재시리오 지하언재시리니 고지즉응하시고 감위수통하소서"

2️⃣ 하괘 결정
   49개 서죽 → 둘로 나눔 → 왼손 취함
   (태극1 + 왼손) ÷ 8 = 나머지 → 선천팔괘 번호

3️⃣ 상괘 결정
   같은 과정 반복 → 64괘 결정

4️⃣ 효 결정
   (태극1 + 왼손) ÷ 6 = 나머지 → 효 위치 (1~6)
```

**선천팔괘 순서**:
```
1:건(乾)☰  2:태(兌)☱  3:리(離)☲  4:진(震)☳
5:손(巽)☴  6:감(坎)☵  7:간(艮)☶  8:곤(坤)☷
```

**영상 컨셉**: "팔괘의 우주적 탄생"

**시각 요소**:
| 요소 | 설명 |
|------|------|
| 배경 | 태극에서 시작되는 우주 창조 장면 |
| 주요 오브젝트 | 거대한 태극 문양 (중앙, 회전) |
| 동작 | 태극에서 8개 괘상(☰☱☲☳☴☵☶☷) 탄생 |
| 효과 | 8괘가 원형 배열 → 빛의 선으로 연결 |
| 특수효과 | 최종 1개 효 선택 시 빛기둥 효과 |
| 색감 | 흑백 태극 + 8색 괘상 + 금빛 선택 효과 |

**영상 사양**:
- 길이: 20-40초 (심리스 루프)
- 해상도: 1920x1080 / 1080x1920
- 형식: MP4/WebM
