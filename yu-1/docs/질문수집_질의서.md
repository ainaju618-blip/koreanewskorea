## 📁 데이터 처리 계획

### 수집 후 처리
1. 질문 텍스트 정제 (오타, 비속어 필터링)
2. 카테고리 자동 분류 (RAG 매칭 테스트)
3. 키워드 추출 → category_seed.py 반영
4. 테스트 케이스 추가 → test_rag_matching.py 보강
5. 해석 데이터 생성 → interpretations_seed.py 확장
