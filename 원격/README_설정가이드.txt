============================================================
   Korea NEWS 원격 스크래퍼 서버 설정 가이드
============================================================

이 문서는 24시간 상시 PC에서 뉴스 스크래퍼를 자동 실행하기 위한
설정 방법을 안내합니다.


[필요한 것]
------------------------------------------------------------
1. Windows PC (24시간 켜져 있어야 함)
2. 인터넷 연결
3. Python 3.10 이상 설치


[설정 순서]
------------------------------------------------------------

1단계: 프로젝트 폴더 복사
   - 이 koreanews 폴더 전체를 원격 PC로 복사
   - USB 또는 압축 파일로 전송

2단계: Python 설치 (없는 경우)
   - https://www.python.org/downloads/ 에서 다운로드
   - 설치 시 "Add Python to PATH" 체크 필수!

3단계: 자동 설정 실행
   - remote_setup 폴더로 이동
   - setup_scheduler.bat 파일을 더블클릭
   - 자동으로 패키지 설치 + 스케줄러 등록됨

4단계: 테스트
   - 명령 프롬프트(cmd) 열기
   - 다음 명령 실행:

     cd [프로젝트 경로]\remote_setup
     python run_all_scrapers.py --dry-run

   - "테스트" 모드로 실행되어 DB 저장 없이 테스트


[자동 실행 스케줄]
------------------------------------------------------------
스케줄러가 등록되면 매일 자동 실행됩니다:
   - 09:00 - 오전 뉴스 수집
   - 13:00 - 오후 뉴스 수집
   - 17:00 - 저녁 뉴스 수집


[수동 실행 방법]
------------------------------------------------------------
필요할 때 수동으로 실행하려면:

# 기본 실행 (오늘 기사만)
python run_all_scrapers.py

# 최근 3일 기사 수집
python run_all_scrapers.py --days 3

# 특정 지역만 실행
python run_all_scrapers.py --regions gwangju naju mokpo


[문제 해결]
------------------------------------------------------------

Q: "Python을 찾을 수 없습니다" 오류
A: Python 설치 시 "Add to PATH" 옵션을 체크하세요.
   또는 PC 재부팅 후 다시 시도하세요.

Q: "모듈을 찾을 수 없습니다" 오류
A: 다음 명령으로 패키지 재설치:
   pip install playwright python-dotenv requests cloudinary supabase
   playwright install chromium

Q: 스케줄러가 작동하지 않음
A: Windows 작업 스케줄러에서 확인:
   - 시작 메뉴 → "작업 스케줄러" 검색
   - "KoreaNews_Scraper_0900" 등의 작업 확인


[파일 구조]
------------------------------------------------------------
remote_setup/
├── .env.production      → 환경변수 템플릿 (프로젝트 루트에 .env로 복사)
├── run_all_scrapers.py  → 전체 스크래퍼 실행 스크립트
├── setup_scheduler.bat  → 자동 설정 스크립트
└── README_설정가이드.txt → 이 문서
